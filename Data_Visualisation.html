Scatter Plot

So grficos que ajudam a analisar a relao entre variveis.
Geralmente colocamos no eixo y a varivel que queremos medir em relao ao eixo x.

Grfico Simples

ggplot(data = mpg, mapping = aes(x = displ, y = hwy)) + 
  geom_point()


Inserir um novo campo como um level no dado, ajuda a interpretar melhor.
O campo class como cor.

# Cor
ggplot(data = mpg, mapping = aes(x = displ, y = hwy, color = class)) + 
  geom_point()


Usar o ?geom_point pode ajudar a configurar melhor a visualizao.


Outra forma interessante de analisar os dados criar facets, que so como camadas com novos grficos, separando conforme um level especfico.

ggplot(data = mpg) + 
  geom_point(mapping = aes(x = displ, y = hwy)) + 
  facet_wrap(~ class, nrow = 2)


Ou com mais de um level.

ggplot(data = mpg) + 
  geom_point(mapping = aes(x = displ, y = hwy)) + 
  facet_grid(drv ~ cyl)

Grfico Smooth

Esse grafico tambm bom para ver a relao entre dua variveis.
Quanto maior a sombra (intervalo de confiana), maior a incerteza dos dados naquele ponto.

Para analisar a relao entre duas variveis temos uma varivel dependente (y), e uma independente (x). Em visualizaes que usam regresso entre as duas variveis geralmente a frmula y ~ x, ou seja, y em funo de x.

ggplot(data = mpg, mapping = aes(x = displ, y = hwy)) + 
  geom_smooth()
## `geom_smooth()` using method = 'loess' and formula = 'y ~ x'

Podemos criar umm grfico com mais de uma linha suavizada.

ggplot(data = mpg, mapping = aes(x = displ, y = hwy, linetype = drv)) + 
  geom_smooth()
## `geom_smooth()` using method = 'loess' and formula = 'y ~ x'

Ou podemos mesclar mais de um tipo de grficos.

ggplot(data = mpg, mapping = aes(x = displ, y = hwy)) + 
  geom_smooth() +
  geom_point()
## `geom_smooth()` using method = 'loess' and formula = 'y ~ x'

Pode alterar os levels do grfico de forma geral para todos, ou se passar argumentos individualmente.

ggplot(data = mpg, mapping = aes(x = displ, y = hwy)) + 
  geom_point(mapping = aes(color = class)) + 
  geom_smooth()
## `geom_smooth()` using method = 'loess' and formula = 'y ~ x'

Grficos de Barra

Grficos de barra mostrar a contagem de ocorrncia em cada categoria da varivel escolhida.

ggplot(data = diamonds, mapping = aes(x = cut)) + 
  geom_bar()

Porm tem como deixar o grficco de barra mais atrativo para anlise.

ggplot(data = diamonds, mapping = aes(x = cut)) + 
  geom_bar(mapping = aes(fill = clarity))

Ou fazendo a anlise em um grfico 100%, ou individal para cada categoria.

#100%
ggplot(data = diamonds, mapping = aes(x = cut)) + 
  geom_bar(mapping = aes(fill = clarity), position = "fill")

#Individual para cada Categoria
ggplot(data = diamonds, mapping = aes(x = cut)) + 
  geom_bar(mapping = aes(fill = clarity), position = "dodge")
